Audio server project given by Amalitech and completed by Tuffour Boateng an Amalitech Trainee.

This project was done according to the requirements of my supervisor.
It features includes:
Security features- Login and Sign up form
music platform where users can play, download and upload music
It has a home section, dashboard section, music section, upload section, about section and contact section.
It made use of node libraries that fits the requirements of this project.
Its database configuration was done through postgresql.

To look at the various libraries click on the package.json file

Install dependencies by typing:
npm install.

To run the program type:
npm run dev.
